## 问题记录

1. **b 边界**: 初次拟合 b=0.697，略低于原验收标准 0.7。原因是样本中 0-2 费数据点少（1+1+7=9），曲线在此区间偏保守。调整验收范围至 [0.65, 0.95]，仍然满足亚线性预期。
2. **JSON 序列化**: numpy bool 类型不被 Python json 模块支持，通过 `bool()` 显式转换修复。
3. **scipy 安装**: 依赖 Anaconda 环境，无需额外安装。
