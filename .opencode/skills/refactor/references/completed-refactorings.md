## 重构记录模板

每次重构完成后，按以下格式追加：

```
## R<N>: <标题>

- **状态：** ✅ 完成 / 🟡 进行中 / ❌ 已回滚
- **目标：** `<目标文件路径>`
- **代码行数变化：** <+N/-M>
- **问题：** <为什么需要重构，1-3 句话>
- **方案：** <做了什么，1-3 句话>
- **教训：** <从这个重构中学到了什么>
- **测试：** <N> 通过 / <M> 失败
```
